Логотип

Логотип проекта представлен в двух вариантах: полный логотип с текстом и иконка.

Цветовая палитра

Проект использует минималистичную цветовую схему, основанную на оттенках синего и серого.

  • Основной цвет: oklch(76% 0.1 228deg) - синий
  • Текст: темно-серый и черный
  • Фон: белый и светло-серый

Типографика

Для основного текста используется шрифт Nunito Sans, который обеспечивает хорошую читаемость. Для примеров кода применяется моноширинный шрифт Source Code Pro.

CSS-фреймворк

В проекте используется CSS-фреймворк Spectre.css, который предоставляет базовые компоненты и утилиты для создания адаптивного интерфейса.

Принципы дизайна

  • Чистота и минимализм
  • Читаемость контента превыше всего
  • Адаптивность для всех устройств
  • Доступность для всех пользователей
