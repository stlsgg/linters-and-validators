О проекте

Причины создания

Проект создавался мной как работа в рамках дисциплины "Веб-технологии" в институте.

По ТЗ, я должен был создать сайт на тему линтеров и валидаторов в веб-разработке, но от себя я еще решил добавить форматтеры. Помимо этого, я создал отдельную страничку со списком полезных ресурсов. Данная страница не имеет прямого отношения к тематике сайта, но может быть полезна начинающим разработчикам.

Если вы начинающий веб-разработчик, возможно, сайт будет полезен для вас. Список полезных ресурсов по веб-разработке

Цели проекта

Создать ресурс, кратко описывающий вспомогательные технологии в веб-разработке, а именно:

  • Линтеры
  • Валидаторы
  • Форматтеры

Для кого это нужно

Сайт прежде всего предназначен для русских разработчиков, которые только-только начинают познавать веб-разработку. Проект не нацелен ответить на все вопросы, но поможет изучить инструменты, которые используются разработчиками со всего мира повседневно.

Стек технологий

Предисловие

Когда я выбирал, с помощью чего я буду писать проект, я не удосужился проверить, можно ли запустить react app в github pages. Поэтому изначально я выбрал простой html, css, js с сборщиком vite. Vite я только изучаю. Я знаю, что есть webpack и что он, как я слышал, является классикой среди сборщиков; но ради простоты и чтобы хоть с чего-то начать, я взял vite.

Только через некоторое время я узнал, что я бы мог развернуть сайт на React через Github Pages, но решил, что не буду переделывать проект под библиотеку. Вместо этого я напишу несколько своих "ухищрений" - не ради оценок или какого-то фидбека от окружющих, - хочу это сделать в рамках изучения, поиграться с JavaScript и инструментами других людей.

Лист использованных технологий

  • HTML
  • CSS
  • Sass - CSS препроцессор
  • CSS фреймворк spectre.css
  • JavaScript
  • Vite - сборщик проекта
  • Animejs - библиотека для анимаций
