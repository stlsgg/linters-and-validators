О проекте

Причины создания

Изначально проект создавался мной как работа в рамках дисциплины "Веб-технологии" в институте, но со временем перерос в нечто большее.

Помимо того, что мне нужно сдать работу, я загорелся желанием переделать этот сайт из справочника по основным линтерам и валидаторам в мини-гайд для начинающих. Помимо линтеров и валидаторов кода я постарался добавить еще немного информации о том, что знаю, в надежде что это будет полезно другим.

Если вы не знаете, как начать путь веб-разработчика, этот сайт не совсем подходит для вас. Здесь описаны лишь некоторые инструменты для создания веб-приложений.

Автор проекта создал список ресурсов, полезный для начинающих разработчиков. От вас лишь требуется уметь читать на английском.

Список полезных ресурсов для изучения веб-разработки

Цели проекта

Создать ресурс, кратко описывающий вспомогательные технологии в веб-разработке, а именно:

  • Линтеры
  • Валидаторы
  • Форматтеры

Для кого это нужно

Сайт прежде всего предназначен для русских разработчиков, которые только-только начинают познавать веб-разработку. Проект не нацелен ответить на все вопросы, но поможет изучить инструменты, которые используются разработчиками со всего мира повседневно.

Стек технологий

Предисловие

Когда я выбирал, с помощью чего я буду писать проект, я не удосужился проверить, можно ли запустить react app в github pages. Поэтому изначально я выбрал простой html, css, js с сборщиком vite. Vite я только изучаю. Я знаю, что есть webpack и что он, как я слышал, является классикой среди сборщиков; но ради простоты и чтобы хоть с чего-то начать, я взял vite.

Лист использованных технологий

  • HTML
  • CSS
  • JS
  • Vite (сборщик проекта)
  • Sass
  • CSS фреймворк spectre.css
